● 数値入力中は、途中の値を勝手に補正しません。Enterまたはフォーカス外れで確定します。
● 各項目は、数値入力とスライダーの両方で操作できます。
● 初期値は、外径150mm・レッグ幅5mm・4本・巻角度300deg・指数1・開始位置0です。
● 画面表示は自動縮尺します。DXFは入力した mm 値そのままで出力します。
● レッグ本数 N を指定すると、360度/N 間隔で等分配置します。
● 青が編集対象の1本、灰色が複製レッグです。
● 終点 P1 をドラッグすると、外径円上で終点角度が変わります。
● DXFは、中心線とレッグ外形の折れ線近似を出力します。
● point curve .txt は、レッグ外形線（閉じたスプライン）のみを出力します。DesignSpark Mechanical の「挿入 → ファイル」で読み込み、カーブ選択 → 「塗りつぶし」 → 「プル」でソリッド化できます。
